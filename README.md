# Customer-Risk-Analysis
This project focuses on identifying customers at high risk of churn and analyzing contributing factors to improve customer retention strategies using Power BI, DAX, and Python.

Project Overview
Customer churn is a critical issue for subscription-based businesses. This project analyzes customer behavior to:

. Detect at-risk customers

. Identify key churn drivers

. Visualize risk levels and service usage

. Recommend data-driven retention strategies

 Key Insights:
 
. Calculated churn rate and service-wise churn breakdown

. Identified customer profiles with high churn probability

. Segmented users based on service usage (TechSupport, Streaming, etc.)

. Visual KPIs on partner status, senior citizens, and online security

. Recommendations for improving customer retention
